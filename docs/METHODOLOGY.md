# 共振 · Resonance — 测评方法论

## 设计目标

Duolog 不是一个"匹配度预测工具"，而是一个**结构化对话触发器**。它的目标是：

1. 帮助两个人系统性地发现自己在关键生活维度上的真实倾向
2. 让分歧可见、可讨论，而不是预测未来
3. 避免社会期望偏差（socially desirable responding）

## 理论框架

### 维度选择

10个测量维度基于以下研究领域：

- **Gottman Institute 的关系研究**：识别高稳定关系的预测因素，包括冲突处理方式、修复机制、双方对关系的感知
- **价值观一致性研究**（Rokeach Value Survey, Schwartz Value Theory）：价值观分歧是长期关系满意度的重要预测变量
- **金钱与关系**：金钱冲突是离婚/分手的主要原因之一（Dew, Britt & Huston, 2012）
- **养育意愿**：关于是否要孩子的分歧属于核心不相容（deal-breaker）类别
- **依恋理论**（Attachment Theory）：情绪成熟度和情感支持能力与关系质量高度相关

### 评分逻辑

每道题的选项被映射到 1–10 的分值范围，代表某个连续维度的两端。例如：

- **金钱观**：1 = 及时享乐、高消费倾向；10 = 储蓄/投资优先、延迟满足
- **冲突处理**：1 = 回避冲突、情绪压抑；10 = 主动面对、修复导向

**分值本身不代表"好"或"坏"**，只代表在该维度上的位置。兼容性通过两人分值的差异（diff）来体现。

### 加权机制

**题目权重（question weight）**

- 默认权重 1.0
- deal-breaker 类题目（如"是否要孩子"）标记为 weight: 2.0
- Deal-breaker 题目在随机抽题时**永远不会被跳过**

**维度权重（dimension weight）**

| 维度 | 权重 | 理由 |
|------|------|------|
| 养育观 | 1.5 | 关于孩子的分歧通常不可妥协 |
| 价值观、金钱观、冲突处理 | 1.3 | 高预测效度的核心维度 |
| 情绪成熟度、亲密与边界、人生观 | 1.1–1.2 | 重要但有协商空间 |
| 成长兼容性、生活合作 | 1.0 | 重要但适应性较强 |
| 社交节奏 | 0.9 | 可通过生活安排协商 |

### 兼容性等级

最终分数 = 各维度加权差异的加权平均值：

```
avgWeightedDiff = Σ(|avgA - avgB| × dimWeight) / Σ(dimWeight)
```

| 等级 | 情侣版 avgWeightedDiff | 已婚版 avgWeightedDiff |
|------|----------------------|----------------------|
| 高度兼容 / 运转顺畅 | ≤ 1.2 | ≤ 1.2 |
| 基本兼容 / 有待磨合 | ≤ 2.2 | ≤ 2.2 |
| 需要深入沟通 / 需要主动处理 | ≤ 3.2 | ≤ 3.2 |
| 差异显著 / 亟需深度对话 | > 3.2 | > 3.2 |

## 版本差异

### 情侣/恋人版（couple）

- 问题使用**假设性场景**和**偏好表达**
- 衡量两人在预期和价值观上的对齐程度
- 结果描述聚焦于"未来兼容性"

### 已婚/同居版（married）

- 问题使用**现在时/过去时**，测量**真实行为模式**
- 衡量当前关系中已存在的协调与摩擦
- 结果描述聚焦于"当前关系健康度"和"需要主动处理的领域"

## 局限性

1. **自我报告偏差**：所有问答类工具都存在作答者美化自身的可能
2. **时间点局限**：人的价值观和优先级会随时间变化
3. **语境缺失**：题目无法捕捉关系中的所有复杂性
4. **非诊断工具**：结果不构成心理咨询或法律建议
